====== 7.Операторы ======
Создан Воскресенье 08 декабря 2019

**Унарным** называется оператор, который применяется к одному операнду. Унарный минус
let = -x;

**Бинарным** называется оператор, который применяется к двум операндам. Тот же минус существует и в бинарной форме:
alert( y - x );

Операторы ++ и -- могут быть расположены не только после, но и до переменной.

let counter = 1;
let a = ++counter; // (*)
alert(a); // 2

В строке (*) префиксная форма увеличения counter, она возвращает новое значение 2. Так что alert покажет 2.let counter = 1;

let a = counter++; // (*) 
alert(a); // 1

В строке (*) постфиксная форма counter++ также увеличивает counter, но возвращает старое значение (которое было до увеличения). Так что alert покажет 1.

**Побитовые операторы**

	AND(и) ( & )
	OR(или) ( | )
	XOR(побитовое исключающее или) ( ^ )
	NOT(не) ( ~ )
	LEFT SHIFT(левый сдвиг) ( << )
	RIGHT SHIFT(правый сдвиг) ( >> )
	ZERO-FILL RIGHT SHIFT(правый сдвиг с заполнением нулями) ( >>> )

Оператор **запятая** предоставляет нам возможность вычислять несколько выражений, разделяя их запятой ,. Каждое выражение выполняется, но возвращается результат только последнего
