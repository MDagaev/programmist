====== 6.Преобразование типов ======
Создан Воскресенье 08 декабря 2019

Строковое – . Может быть вызвано с помощью **String(value)**. Для примитивных значений работает очевидным образом.

Численное –  Может быть вызвано с помощью **Number(value)**

Логическое –  Может быть вызвано с помощью **Boolean(value)**
