====== 1.Свойства ======
Создан Воскресенье 08 декабря 2019

**Свойство – это пара «ключ: значение»,** где ключ – это строка (также называемая «именем свойства»), а значение может быть чем угодно

**let user = {     // объект**
**  name: "John",  **// под ключом "name" хранится значение "John"
**  age: 30        **// под ключом "age" хранится значение 30
 **"likes birds": true **// имя свойства из нескольких слов должно быть в кавычках
**};**

обращения к свойствам используется запись «через точку»:
**user.name**

Для удаления свойства мы можем использовать оператор delete:
**delete user.age;**

Для свойств, имена которых состоят **из нескольких слов**, доступ к значению **«через точку» не работает**:

**user.likes birds** // так не работает
**user["likes birds"]**  // так работает

Квадратные скобки дают намного больше возможностей, чем запись через точку. Они позволяют использовать любые имена свойств и переменные, хотя и требуют более громоздких конструкций кода.

Подведём итог: в большинстве случаев, когда **имена свойств известны и просты**, используется **запись через точку**. Если же нам нужно что-то более сложное, то мы **используем квадратные скобки.**
