====== 2.Свойства из переменной ======
Создан Воскресенье 08 декабря 2019

часто нам необходимо использовать существующие переменные как значения для свойств с тем же именем.

**function makeUser(name, age) {**
**  return {**
**    name: name,**
**    age: age**
**    // ...другие свойства**
**  };**
**}**

свойств **name и age** совпадают с **названиями переменных**, которые мы подставляем в качестве значений этих свойств. Такой подход настолько распространён, что существуют специальные короткие свойства для упрощения этой записи.

Вместо name:name мы можем написать просто name:
**function makeUser(name, age) {**
**  return {**
**    name, **// то же самое, что и name: name
**    age  **    // то же самое, что и age: age
**    // ...**
**  };**
**}**
